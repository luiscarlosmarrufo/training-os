# Schema Changelog

## v0.1 - Initial Schema (2026-02-24)

### Added
- **Entity types:** exercise, protocol, progression_model, program, routine, rule, glossary
- **Exercise fields:** Core movement tracking with fatigue and spinal load metrics
- **Protocol fields:** Training methodologies and execution patterns
- **Progression fields:** Progressive overload models
- **Program fields:** Multi-day training structure
- **Controlled vocabulary:** Movement patterns, fatigue levels, spinal load levels

### Rationale
Starting with minimal schema focused on resistance training. Additional fields for nutrition, sleep, recovery metrics will be added when that content appears in inbox.

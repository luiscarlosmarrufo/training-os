# Aliases & Decoding

## Abbreviations
- SS -> superset
- Max -> rest-pause: to failure, rest ~20 sec, continue
- AMRAP -> as many reps as possible
- RIR -> reps in reserve
- RPE -> rate of perceived exertion
- EMOM -> every minute on the minute
- E2MOM -> every 2 minutes on the minute
- DB -> dumbbell
- BB -> barbell
- KB -> kettlebell
- RDL -> romanian deadlift
- SLDL -> stiff leg deadlift
- GHR -> glute ham raise
- BSS -> bulgarian split squat
- FFE -> foot forward elevated (split squat variant)
- OHP -> overhead press
- CGBP -> close grip bench press
- WG -> wide grip
- NG -> neutral grip
- T1, T2, T3 -> tier 1, 2, 3 (exercise priority levels)

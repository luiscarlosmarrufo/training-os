# Dylan Shannon Sprint/Jump/Heavy Program

## Day 1 - Sprint/Jumps/Heavy

### Global Rules
- "Max" = failure → rest 20 sec → continue (rest-pause protocol)
- SS = superset

### A) Conditioning Circuit - Dynamic Warmup

**Movement prep (10 yards each):**
1. Hamstring scoops
2. Quad pull w/ reach
3. Reverse lunge w/ reach
4. High kicks
5. Knee hug → Forward lunge
6. Single-leg RDL (walk)

**Activation:**
- Single switches - 2×10 yds
- Double switches - 2×10 yds
- Triple switches - 2×10 yds
- Straight-leg bounds - 3×20 yds

### B) Strength/Power

#### 1) Sprints (8-week progression)
**Rest formula:** yards ÷ 10 = minutes rest

- **Week 1:** 3×10, 3×15, 4×20
- **Week 2:** 5×15, 5×20
- **Week 3:** 3×15, 4×20, 2×30
- **Week 4:** 3×15, 3×20, 3×30
- **Week 5:** 4×20, 4×30
- **Week 6:** 4×20, 5×30
- **Week 7:** 8×30
- **Week 8:** 6×40

#### 2) Jumps
- Seated vertical jump - 3×3
- Seated broad jump - 3×3

#### 3) Olympic Lifts

**Weeks 1-4:**
- Block Clean - 4×3

**Weeks 5-8:**
- Clean wave: 3/3/2/2/1 (increase weight as reps decrease)

#### 4) Back Squat (8-week progression)

**Phase 1 (Weeks 1-4): Straight Sets**
- W1: 4×8 @ 70-75%
- W2: 4×7 @ 77.5-82.5%
- W3: 5×6 @ 80-85%
- W4: 5×5 @ 80-85%

**Phase 2 (Weeks 5-6): Potentiation Waves**
- W5: 6-4-2 / 6-4-2 @ 80-85%
- W6: Same structure, heavier

**Phase 3 (Weeks 7-8): Heavy Waves**
- W7: 5-3-1 / 5-3-1 @ 85-95%
- W8: Same structure, heavier

### C) Accessories

**Weeks 1-6: Superset ×3**
- DB Bulgarian split squat - Weekly reps: 8, 8, 7, 6, 6, 5
- Glute-ham raise (3-sec eccentric) - Same rep progression

**Weeks 7-8: Superset ×3 (swap)**
- Goblet lateral split squat - 8 reps
- Single-leg calf raise - 10 reps

## Notes
- 8-week progressive overload across all main movements
- Sprint volume and distance increase systematically
- Olympic lifts transition from blocks to full cleans
- Back squat uses three distinct phases
- Accessory work changes in final 2 weeks

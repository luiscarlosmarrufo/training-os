# Main Lift 4-Week Wave

## Pattern
- **Week 1:** 4×6 @ 75%
- **Week 2:** 4×5 @ 77%
- **Week 3:** 5×5 @ 80%
- **Week 4:** 5×4 @ 85%

## Implementation
- Use same lift for 4 weeks
- Follow the wave progression above
- Rest 2-4 minutes between sets

## After Completion
- Option 1: Restart wave slightly heavier (preferred)
- Option 2: Deload 1 week, then restart

## Notes
- Designed for main compound lifts
- Balances volume and intensity
- Progressive overload through both load and volume changes

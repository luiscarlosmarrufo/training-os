# Romanian Deadlift

## Description
Hip hinge exercise starting from top position, focusing on eccentric hamstring loading. Essential posterior chain developer.

## Key Cues
- Maintain slight knee bend throughout
- Push hips back
- Keep bar close to body
- Feel stretch in hamstrings
- Drive hips forward to return

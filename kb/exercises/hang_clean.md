# Hang Clean

## Description
Olympic lift variation starting from hang position. Develops explosive power and full-body coordination.

## Key Cues
- Start with bar at mid-thigh
- Aggressive hip extension
- Pull elbows high and outside
- Catch in front squat position
- Stand to complete

## Common Uses
- Part of complex with Front Squat + Push Press
- Power development for athletes

## Notes
- Requires technical proficiency
- Rest 90-180 seconds between sets for quality
